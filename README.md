# Netflix EDA — CodeAlpha Data Analytics Internship
Exploratory Data Analysis on Netflix Movies and TV Shows dataset.

## Questions Answered
1. Which country produces the most Netflix content?
2. Which year had the most content added?
3. How has the Movies vs TV Shows ratio changed over time?

## Key Findings
- USA dominates with 2818 titles, India second with 972
- 2019 had peak content additions, declined due to COVID-19
- Movies outnumber TV Shows 6131 vs 2676

## Tools Used
Python, Pandas
